## 📋 О Проекте

**SpyderOSINT** - это полнофункциональная система OSINT (Open Source Intelligence) с оркестрацией множественных ИИ моделей для поиска и анализа информации по номерам телефонов.

### Основные Компоненты:

1. **Многоагентная архитектура** - 5 специализированных агентов
2. **Оркестрация ИИ моделей** - поддержка OpenAI, Anthropic, Groq, Ollama
3. **Интеграция 40+ источников** - соцсети, БД, государственные реестры, утечки
4. **Реальные API** - работающие бесплатные и платные сервисы
5. **Многоуровневый анализ** - валидация → поиск → агрегация

## 🤖 Метаданые Agentов

### 1. PhoneValidationAgent
- **Роль:** Phone Validator
- **Функции:** Валидация, нормализация, определение оператора
- **Источники:** Встроенные БД операторов
- **Точность:** 99%

### 2. SocialMediaAgent
- **Роль:** Social Media Searcher
- **Функции:** Генерация ссылок, стратегии поиска
- **Платформы:** 9+
- **Результативность:** Высокая (требует дополнительного ручного поиска)

### 3. DatabaseAgent
- **Роль:** Database Researcher
- **Функции:** Поиск в государственных реестрах
- **Источники:** Росреестр, ФНС, 2GIS
- **Доступность:** Public API

### 4. LeakDatabaseAgent
- **Роль:** Breach Database Researcher
- **Функции:** Проверка компрометированных данных
- **Источники:** HaveIBeenPwned, DeHashed
- **Риск:** Высокая вероятность найти личные данные

### 5. AggregationAgent
- **Роль:** Data Aggregator
- **Функции:** Объединение и анализ
- **Выходные данные:** Полный профиль
- **Формат:** JSON, TXT, HTML
