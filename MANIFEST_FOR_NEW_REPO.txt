🚀 БЫСТРЫЙ СТАРТ ДЛЯ НОВОГО РЕПО
================================================================================

1. Установка:
   ```
   git clone <новый-репо>
   cd spyder-osint
   pip install -r requirements.txt
   ```

2. Использование RealSearchSystem:
   ```python
   from core.real_search_system import RealPhoneSearchSystem
   searcher = RealPhoneSearchSystem()
   report = searcher.generate_full_report("+79182469659")
   filename = searcher.save_report(report)
   ```

3. Использование AgentOrchestrator:
   ```python
   from core.advanced_agent_orchestrator import AgentOrchestrator
   orchestrator = AgentOrchestrator()
   results = orchestrator.execute_all_agents("+79182469659")
   ```

4. Использование Google Dorks:
   ```python
   from core.kali_linux_tools import GoogleDorkEngine
   engine = GoogleDorkEngine()
   dorks = engine.generate_phone_dorks("+79182469659")
   ```

5. Запуск полной системы:
   ```
   python main_osint_orchestrator.py "+79182469659"
   ```
